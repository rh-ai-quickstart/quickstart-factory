# Grooming Criteria for AI Quickstarts

## Scoring Rubric (0–10)

| Score | Readiness | Criteria |
|-------|-----------|----------|
| 0–2 | Raw idea | No owner, no scope, no business outcome |
| 3–4 | Early idea | Owner identified, rough scope, outcome articulated |
| 5–6 | Groomed | Scope clear, dependencies listed, ready for sprint |
| 7–8 | In progress | Repo exists, active development, demo available |
| 9–10 | Done | In catalog, documented, reviewed |

## Readiness Gates

### Idea → In Progress

- [ ] Business outcome documented
- [ ] Owner assigned
- [ ] Scope defined (v1 boundaries)
- [ ] Dependencies identified
- [ ] Repository created or planned

### In Progress → Done

- [ ] Code complete and reviewed
- [ ] Documentation complete
- [ ] Demo video or walkthrough available
- [ ] Catalog entry created
- [ ] Blog post (draft or published)
